A browser is the main window to the entire Web for 4.6 billion people. Chromium (the Open Source version of Chrome), from which many other browsers (like Brave or Microsoft Edge) are heavily derived is owned and led by Google.

Firefox is the only fully open source browser with its own alternative technologies. However its non-profit developer (Mozilla) is mostly funded by Google. Apart from Safari, Google therefore has a very dominant influence on this market.
